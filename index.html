MatTech DOS

A VirtualBox-style DOS and Windows emulator for Ubuntu. Run Windows 1.0 through XP, MS-DOS, and more — with one-click disk swapping and auto-download of install media.

Ubuntu / Debian QEMU-powered GTK3 Python 3 Free & Open

Install after downloading:
sudo apt install ./mattech-dos_1.0.0_all.deb
Or install dependencies manually first:
sudo apt install qemu-system-x86 qemu-utils python3-gi python3-gi-cairo gir1.2-gtk-3.0 xdotool genisoimage

✨ Features

🖥️

VirtualBox-style UI

Left sidebar machine list, right detail panel — familiar layout for anyone who's used VirtualBox.

⚡

QEMU-powered

Full x86 PC emulation via QEMU. Proper hardware emulation, not a DOS compatibility layer.

💾

Auto-download Install Media

Pick a DOS version and click Download — install disks are fetched from archive.org automatically.

🔄

One-click Disk Swap

Floating disk swap panel lets you change floppy and CD-ROM images instantly via QEMU monitor — no typing.

💿

Blank Floppy & CD Creator

Create blank floppy images (FAT12) or blank CD-ROM ISOs with custom volume labels in seconds.

📸

Save & Resume State

VM state is saved automatically on shutdown and resumed on next start — just like VirtualBox snapshots.

⚙️

Auto-detected Requirements

Select a DOS version and the app automatically sets the correct CPU, RAM, and disk size for that OS.

🔁

Factory Reset

Wipe a VM's hard disk and start fresh with one click. Downloaded media is preserved.
